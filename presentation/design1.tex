      \item Modularity

        \item Separated components
        \item Data protocol - loose coupling
        \item Individual component development
      \end{itemize}
      \item Scalability

        \item Resolve bottlenecks

    \item Adhere to protocol

    \end{itemize}
  \end{itemize}
